アプリケーション名　**comrade**

アプリケーション概要
仕事や遊びなど、何かを共にやる仲間を見つけるアプリです。ユーザーは画像付きの投稿で自分のやりたいことを共にしてくれる仲間を集めることができます。また、他ユーザーの投稿を見て仲間に名乗りを上げることができます。

利用方法
ユーザー登録をすると画像つき投稿で仲間を集められるようになります。
他ユーザーのやりたいことに名乗りを挙げられます。
